SORT
=====

A simple online and realtime tracking algorithm for 2D multiple object tracking in video sequences.
See an example [video here](https://motchallenge.net/movies/ETH-Linthescher-SORT.mp4).

By Alex Bewley  
[DynamicDetection.com](http://www.dynamicdetection.com)

### Introduction

This version is based on simple data association from frame to frame (without using Kalman predictions)
